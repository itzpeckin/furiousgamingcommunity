# Franchise HQ — Milestone 1A

This is the first interactive design-system prototype for the Madden league website.

## Included

- Responsive desktop, tablet, and mobile application shell
- Placeholder brand and league identity
- Primary league navigation
- Role preview for Commissioner, Team Owner, and Trade Committee
- League-home dashboard mockup
- Reusable design-system page
- Accent-color preview controls
- Compact and comfortable density modes
- Keyboard quick search with `Ctrl/Cmd + K`
- Browser-local preference storage
- Placeholder routes for all planned league pages

## Open it

Double-click `index.html` or upload the folder contents to a static host such as GitHub Pages.

No installation, build process, or server is required.

## Files

- `index.html` — page structure and prototype content
- `styles.css` — responsive visual system
- `app.js` — navigation and interactive behavior

## Current boundaries

This milestone intentionally uses mock content. It does not yet include the complete team, player, standings, stats, news, or trade workflows. Those pages are represented by connected placeholders so later milestones can be added without rebuilding the shell.
