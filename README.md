# Furious Gaming Community — Franchise HQ

## Sprint TC-001: Player Valuation Experience

This release preserves the approved Franchise HQ design and adds a large interactive player valuation card to trade offers.

### Included
- Click any player in an authorized trade package to open the player card.
- Large modal that closes with X or by clicking the backdrop.
- Four top content boxes: player overview, position-aware key attributes, three-season stats, and contract details.
- Full-width League Valuation Engine breakdown on the third row.
- Simplified fairness panel with totals, fairness percentage, and comparison bar only.
- DEV-001 Developer Mode remains included.

### Mock-data note
Attributes, historical seasons, and draft information are prototype values derived from the current mock roster. Madden export data will replace these fields later.
